🗺️ GeoServer Configuration for School Accessibility

WFS Endpoints (Already Available)

  • /wfs/schools - Schools data in GeoJSON
  • /wfs/sectors - Sectors data in GeoJSON
  • /wfs/roads - Roads data in GeoJSON

GeoServer Setup (Optional)

To use full GeoServer capabilities:

1. Download GeoServer: https://geoserver.org/download/
2. Install and start GeoServer (default: http://localhost:8080/geoserver)
3. Copy GeoJSON files from: ~/Desktop/School_Accessibility_Data/shapefiles/
4. Create new workspace and stores in GeoServer
5. Publish layers as WMS/WFS services

Data Files Location

~/Desktop/School_Accessibility_Data/shapefiles/

  • schools.geojson
  • sectors.geojson
  • roads.geojson

WMS GetCapabilities

http://localhost:8080/geoserver/wms?service=WMS&version=1.3.0&request=GetCapabilities

WFS GetCapabilities

http://localhost:8080/geoserver/wfs?service=WFS&version=1.1.0&request=GetCapabilities
